# Interactive Investment Calculator

A lightweight, browser-based tool to simulate financial growth using compound interest. It allows you to model complex investment scenarios involving different life stages.

## üí° What is this?
This calculator helps you visualize your financial future by answering questions like:
*   *"How much will I have in 30 years if I invest $500 monthly?"*
*   *"What happens if I stop contributing and just let the interest grow?"*
*   *"How long will my savings last if I start withdrawing funds?"*

## ‚ú® Key Features
- **Multi-Phase Planning:** Combine different periods of **Depositing** (saving), **Waiting** (compound interest only), and **Withdrawing** (spending).
- **Age Tracking:** Optional input to track how old you will be at the end of each investment phase.
- **Dynamic Chart:** Visualizes your Total Balance vs. Principal Investment in real-time.
- **Privacy Focused:** Runs entirely in your browser; no data is sent to any server.
